# 本地开发环境脚本

本目录包含用于启动和管理AI4SE工具集本地开发环境的脚本。

## 主要脚本

### 1. `start-local-dev.sh`
详细的环境检查和准备脚本，包含：
- Python环境检查
- 虚拟环境检测和激活
- 依赖检查
- 数据库状态检查

## 使用方法

### 快速启动
```bash
# 在项目根目录执行
./dev.sh
```

### 查看帮助
```bash
./dev.sh --help
```

### 常用命令
```bash
./dev.sh start          # 启动开发环境
./dev.sh stop           # 停止所有服务
./dev.sh restart        # 重启服务
./dev.sh status         # 查看服务状态
./dev.sh logs           # 查看实时日志
./dev.sh health         # 健康检查
./dev.sh clean          # 清理环境
```

## 解决的问题

1. **端口占用问题** - 自动检测并清理占用的端口
2. **进程管理** - 智能管理Flask进程，避免僵尸进程
3. **启动卡顿** - 包含健康检查，确保服务完全启动
4. **日志管理** - 统一的日志输出和查看
5. **环境清理** - 一键清理所有相关进程和临时文件

## 配置选项

- `--port PORT`: 指定Flask端口（默认5001）
- `--no-browser`: 不自动打开浏览器
- `--verbose`: 详细输出模式

## 故障排除

### 服务无法启动
1. 检查端口占用：`./dev.sh status`
2. 查看日志：`./dev.sh logs`
3. 清理环境：`./dev.sh clean`

### 页面加载慢
1. 运行健康检查：`./dev.sh health`
2. 重启服务：`./dev.sh restart`

### 端口被占用
脚本会自动处理端口占用问题，强制清理占用的进程。
