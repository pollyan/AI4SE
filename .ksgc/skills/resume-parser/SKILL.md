# Resume Parser Skill

## 概述

Resume Parser 是一个专业的简历解析工具，能够从各种格式的简历文件（PDF、Word、TXT）中提取结构化信息。

## 功能特性

- **多格式支持**: PDF、DOCX、DOC、TXT
- **智能解析**: 支持多种简历格式和布局
- **双重策略**:
  - **LLM 解析**: 使用当前对话的 LLM 智能解析（推荐，准确率最高）
  - **正则表达式**: 本地解析，无需外部 API（适合简单格式）
- **结构化输出**: JSON 格式，易于集成

## 提取字段

| 字段 | 说明 |
|------|------|
| 姓名 | 候选人姓名 |
| 性别 | 男/女 |
| 年龄 | 整数 |
| 学历 | 最高学历（本科、硕士、博士等） |
| 工作年限 | 浮点数 |
| 出生年月 | YYYY年MM月 或 YYYY-MM |
| 教育经历 | 数组，包含学校、学历、专业、开始时间、结束时间 |
| 工作经历 | 数组，包含公司、职位、开始时间、结束时间、工作描述 |

## 使用方法

### 激活 Skill

在对话中激活此 Skill 后，可以直接发送简历文件或文本进行解析。

### 示例

**解析 PDF 简历**：
```
请解析这个简历文件：/path/to/resume.pdf
```

**解析 Word 简历**：
```
请帮我提取这个简历的信息：/path/to/resume.docx
```

**解析文本简历**：
```
请解析以下简历内容：
【简历文本内容】
```

## 解析策略

当激活此 Skill 后，解析流程如下：

1. **提取文本**: 使用 PyPDF2 或 python-docx 从文件中提取文本
2. **LLM 解析**: 使用当前对话的 LLM 智能解析简历内容
3. **结构化输出**: 返回 JSON 格式的结构化数据

## 依赖安装

```bash
pip install PyPDF2 python-docx
```

## 支持的简历格式

LLM 解析可以处理任何格式的简历，包括：
- 非标准布局
- 手写体（OCR 后）
- 多语言简历
- 复杂格式
- 各种模板（智联招聘、前程无忧、BOSS直聘等）

## 常见问题

### Q: 支持哪些语言？

A: 主要支持中文简历，也可以处理英文简历。

### Q: 如何处理大量简历？

A: 可以批量发送简历文件进行解析，或者编写脚本调用解析功能。

### Q: 解析准确率如何？

A: 使用 LLM 解析准确率很高，可以处理各种复杂格式的简历。
